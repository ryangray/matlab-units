To install, change to this folder and give the command:

	units install

You will be prompted to save the changes to the path.

To manually install, add this folder and the si\ subfolder
to your MATLAB path. To hook up the help browser, click on 
the MATLAB Start button and choose:
	Desktop Tools -> View Start Button Configuration Files...
Then click "Refresh Start Button", then click "Close".
The units toolbox should appear in the help browser tree.
You can use the commands "doc units" or "units help" to bring
up the documentation.
